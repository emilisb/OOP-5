#  5 užduotis

![version][version-badge]

## [Užduoties sąlyga](https://github.com/objprog/paskaitos2019/wiki/5-oji-užduotis)

## Parsisiuntimas (Unix platformoje)

- `git clone https://github.com/emilisb/OOP-5.git`
- `cd OOP-5`

## Kompiliavimas

- `make`

## Paleidimas

- `make run`

**arba**

- `cd bin`
- `./uzduotis`
